# Laravel Authentication Project

A simple user authentication application built with Laravel. This project demonstrates user registration, input validation, password hashing, session-based authentication, and logout functionality.

## Features

- User registration
- Server-side form validation
- Unique username and email validation
- Secure password hashing
- Automatic login after registration
- Session-based authentication
- User logout
- Blade templates for the user interface
- Eloquent ORM for database operations

## Tech Stack

- PHP 8.3
- Laravel 13
- Blade
- Eloquent ORM
- SQLite
- Vite
- HTML and CSS
